Principais formatações

Podemos criar tambem um texto marcado usando a tag <mark>.

E no outro parágrafo, temos outro texto marcado no final.

Texto grande e pequeno

Estamos criando um texto grande com a tag <big> e um texto pequeno com a tag <small>

Texto deletado

Podemos marcar um texto como excluído com a tag <del>,para indicar que ele deve ser lido mas não considerado.

Texto inserido

Podemos marcar um texto como inserido com a tag <ins>,para dar enfâse e indicar que ele foi adicionado depois.

Existe também o sublicado com a tag <U> (não semantica)

Texto sobrescrito

Para inserir coisas do tipo x8 + 3, usando a tag <sup>.

Texto subscrito

Para inserir coisas do tipo H2O, usando a tag <sub>.
